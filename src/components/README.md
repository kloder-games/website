# Components Organization

## /\<page-name\>/\*

Components that are created for one page only, for example:

- Grid on games
- Item on games
- Slider on game
- ...

## /layout/\*

Components that are part of any layout, for example:

- Footer
- Header
- LanguageSwitcher
- ...

## /\*

Generic components ready to be used on any place, for example:

- Button
- Social
- Separator
- Icon
- ...
